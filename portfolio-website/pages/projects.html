Projects


  • Project #1
For this project I created a concert poster for Lauryn Hill and the 20th anniversary tour for her album, The Miseducation of Lauryn Hill. Her album artwork for The Miseducation of Lauryn Hill, features a photo of the top of a school desk. I wanted to showcase a mix of the old artwork in a new and modern way, as it was the 20th anniversary tour.
  • Project #2
For this project I created an album color for the band The XX, I wanted to keep it simple yet aesthetically pleasing. I wanted to create a visual expression of their music which is euphoric and ethereal, while also keeping it minimal. The background image was made by combining several images, the multiple layers created the shapes that ultimately make up the colorful background image.
  • Project #3
This project focused on using simple lines and shapes to create the center image, a radio. I wanted to give an example of different variations of an image that can be created by only using lines or circles. It was important for me to showcase my creativity even under specific conditions such as restricting myself to specific shapes, to show my flexibility in my design process.
  • Project #4
This project is an example of my layout design work. In this project I used hierarchy to showcase the important aspects of the layout design. While using all the needed information, I wanted to present it in a clear yet interesting manner.
